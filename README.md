# Bloom Baby Tracker

A calm offline-first progressive web app for logging:

- sleep
- breastfeeding and bottle feeds
- diaper changes
- weight and length measurements

## Features

- Multi-baby profiles
- IndexedDB storage in the browser
- Export and import for migration between browsers
- Installable PWA with offline support
- Responsive, pastel UI tuned for iPhone home-screen use

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
